Divers : 

Un Philosophe peut faire trois choses :  
  - Manger
  - Dormir
  - Penser

Pour manger il doit choisir deux fourchettes (celle devant lui et une autre a sa droite ou a sa gauche.
Le nombre de fourchette est egal au nombre de philosophes.
